# Code of Conduct

Esperamos colaboração respeitosa, objetiva e profissional em todos os canais.

## Comportamentos esperados

- Comunicação clara e respeitosa
- Feedback técnico com foco em melhoria
- Abertura para revisão por pares

## Comportamentos inaceitáveis

- Assédio, discriminação ou ataques pessoais
- Exposição indevida de dados sensíveis
- Ações que violem políticas de segurança e conformidade

## Reporte

Reporte violações para o time Platform/Security pelos canais internos oficiais.
